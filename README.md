## Como correr la práctica:
-Activar el entorno virtual con: $ source env/bin/activate

-Usando Cursores, correr el script cursores.py con: 

$ python3 cursores.py

-Usando Alchemy, correr el script app.py con: 

$ python3 Alchemy/app.py
